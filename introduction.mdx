NvChad is a Neovim framework that transforms your editor into a powerful, IDE-like environment without sacrificing speed. Built entirely in Lua, it combines a carefully curated set of plugins with custom UI components and intelligent defaults — giving you a stunning, productive Neovim setup in minutes.

## Why NvChad?

NvChad was built for developers who want a beautiful, fast Neovim experience without spending days configuring it from scratch. It's designed to be fast first — plugins are lazy-loaded by default, keeping startup times around 0.02–0.07 seconds even on low-end hardware.

## Key Features

- **56+ color themes** with a live theme switcher — no restart required
- **Custom UI components**: statusline, tabufline (buffer manager), and NvCheatsheet
- **LSP out of the box** via `mason.nvim` and `nvim-lspconfig` for code intelligence
- **Telescope** for fuzzy finding files, buffers, live grep, and more
- **Treesitter** for accurate, fast syntax highlighting across dozens of languages
- **nvim-cmp** autocompletion with snippet support via LuaSnip
- **Git integration** through gitsigns.nvim
- **File tree** via nvim-tree.lua
- **Lazy loading** on 93% of plugins for near-instant startup
